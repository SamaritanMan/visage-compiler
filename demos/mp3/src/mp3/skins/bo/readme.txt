1. INTRODUCTION
thank you for downloading this skin. make sure you have at least 
winamp 2.09, otherwise you are not able to use this skin as your 
winamp interface. check www.winamp.com for the latest release of 
winamp.

2. INSTALLATION
unzipping to your winamp/skins directory will create a subfolder 
storing all data. launch winamp, go to the option-menu and use 
the skin-browser to select the skin. for users of winamp 2.04 or 
higher it's enough to copy the zip-file to the winamp/skins directory.
