This document covers the JavaFXTM Script language, API, and tools. All aspects of the JavaFX Script language are undergoing rapid development, so the documentation referenced here is all draft quality which may change at any time. This draft documentation is to assist early evaluators of this changing technology, and does not make any commitment to how the platform will be defined when it is first released.
