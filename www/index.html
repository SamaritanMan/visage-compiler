What is the Visage Compiler Project

The Visage Compiler project is an Visage incubator project. It focuses on creating a Visage compiler to translate Visage files into JVM class files (bytecode). This compiler leverages and extends the JDK's javac compiler capabilities and, hence, benefits from all of the engineering efforts that have already been put into javac.

This project is open so the Visage community members can participate in its design and implementation. We welcome participation by allowing the Visage community members to browse the source code, review and address issues, contribute to the Wiki, and join the discussion aliases. Together we can define and develop a first-class compiler and runtime for the Visage language.

In February 2009, the source code for this project moved from a Subversion repository at this site to a Mercurial repository at kenai.com. For the time being, this java.net site and the associated discussion groups will continue to be used for project content other than the actual source code. Please see Using kenai.com and Mercurial for more information about this transition.

How to Contribute

You are invited to join the project, download the source code, contribute, or comment on and provide feedback on the Visage compiler as it continues to be developed and released.

  • Read the process to contribute.
  • Visit the Visage Compiler project at kenai.com.
  • Review the issues database for the current list of compiler and Visage language issues to be resolved
  • View the project's Wiki page, which will have ongoing summaries of the various decisions that went into the project
  • Click Announcements to view announcements and news relevant to this project.
  • Click Discussion Forums to start or join a discussion on general or specific Visage Compiler topics, or to share announcements relevant to this project. (While anyone can view the site resources, including the forums, in order to POST or REPLY in the forums both java.net membership and the Observer role in Visage Compiler project are required. Registering for a java.net user account can be done here, then the Observer roles in Visage Compiler project can be requested here).
  • Join the project's Mailing Lists
  • You can also send an email to the project administrator.
