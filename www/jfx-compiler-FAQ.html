Where can I download the Visage compiler?

The Visage compiler is in a Mercurial respository in the Visage Compiler project at kenai.com, and can be downloaded like this
hg clone http://kenai.com/hg/visage-compiler~marina-master

Where is the Visage language specification?

The Visage language specification is very incomplete at this point, and not useful as a reference yet. To build an HTML version of the current draft, download the project's source code, then go to the doc/ directory and run "ant spec".

What is the relationship between javac (JDK's Java compiler) and the Visage compiler?

The Visage compiler extends the javac compiler, providing a new lexer, parser, syntax trees, etc.  This is done to leverage the engineering years of effort that has already been invested in javac.

What are the open Visage language issues?

They are listed in the Visage Compiler project's issue database, under the "specification" subcomponent.  The list is incomplete, but is growing as we uncover new areas that need specification.

What discussion has there been regarding these language issues?

The Visage Compiler project's wiki page contains notes from our engineering meetings, which we will expand upon. We welcome developer comments regarding all language and compiler issues, either via the dev@visage.org email alias, or in the wiki.

When debugging or profiling the compiler, how do I reference javac sources?

First, you need a copy of the OpenJDK source code, which contains the sources. The "javac" NetBeans project in the Visage compiler source base is then configured to reference that local OpenJDK directory. Full instructions are found in the javac/README file.
